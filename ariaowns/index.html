Accordions with and without aria-owns

Instructions:

  1. Open NVDA
  2. Tab to each accordion button and activate it to expand the accordion
  3. Tab to the first control in the expanded accordion (activating it will fire an alert)
  4. Tab back to the parent expand/collapse button and listen to what NVDA reads.
  5. Activate the parent expand/collapse button and observe that on the one with aria-owns, the button in the expandable section gets clicked.

Behavior

Expected

Tabbing into the expanded accordion, then back to the expand/collapse button, and reactivating the button, should collapse the accordion. The button inside the accordion should not get clicked.

Shift-tabbing back to the expand/collapse button should cause NVDA to read the button's label.

Actual

Tabbing into the expanded accordion, then back to the expand/collapse button, and reactivating the button, causes the button inside the accordion to get clicked.

Shift-tabbing back to the expand/collapse button causes NVDA to not read the button's label.
